# BACKEND SETUP
## Create environment variable

Copy environment variables from `.env.example` to `.env`

## Run the application

We are using `npm` as a package manager

### Install dependencies

```
npm install
```

### Run Server

```
npm run dev
``` 

## Building application

```
npm run build
```
